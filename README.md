A simple task management and focus tool.

# ✨ Features

- Focus Presets (swap between timers for different tasks quickly)
    - Comes with 12 presets out of the box
    - Ability to create your own presets
- 10 unique focus sounds
- Reward system for additional break time
- Transition periods
- Clean and simple interface
- 10 different themes
- Discord Rich Presence support (to let your friends know you're busy)
- Super configurable
